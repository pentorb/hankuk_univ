# 🏫 한국대학교 포털
KOSTA 최종 프로젝트 (2024.05 - 2024.06)
<br><br>

## 📌 프로젝트 개요

- 학사포털과 온라인 클래스를 하나로 합친 웹 어플리케이션

<br>

## 📋 주요 기능

### 👨‍🎓 학생
- 휴학, 복학 신청
- 졸업관리
- 성적조회
- 성적 이의신청
- 과제제출
- 공결신청
- 수강신청

### 👩‍🏫 교수
- 강의계획서 등록
- 과제 등록
- 시험문제 등록
- 강의자료 등록
- 학생 출결관리
- 학생 성적관리

### 👨‍💼 교직원
- 계정관리
- 학과관리
- 휴학관리
- 강의등록

<br>
